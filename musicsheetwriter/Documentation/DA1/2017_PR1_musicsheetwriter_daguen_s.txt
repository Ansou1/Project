Application Windows Phone:
	- Un device (smartphone) avec Windows Phone
	- La version de Windows Phone est au minimum 8.1
	- Une connexion internet
	- La partie API du site web doit être installée.
	- Permissions requises:
		- Accès à internet
		- Ecriture dans la memoire du telephone
		- Appareil photo
		- Gestion des comptes enregistrés sur le telephone
